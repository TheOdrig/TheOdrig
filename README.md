# Mehmet Akif Uludag

Software Engineering student focused on production-grade backend and full-stack development with Java, Spring Boot, PostgreSQL, Next.js, React, and TypeScript.

I am currently building private production software and keeping that source code off my public GitHub profile. My public repositories are proof projects that show my backend, frontend, testing, and architecture style.

## What I Build

- **Private production work**: Java/Spring Boot and Next.js full-stack systems with modular backend architecture, secure authentication, search, CI/CD, observability, and SEO-oriented product work.
- **Car Rental System**: public full-stack proof project with a Spring Boot + Spring Modulith backend and a Next.js + TypeScript frontend using Feature-Sliced Design and the API Proxy pattern.
- I care about modular boundaries, secure authentication, database ownership, testable business flows, and turning real product metrics into engineering decisions.
